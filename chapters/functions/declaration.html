Объявление функций в C++

Функции - это основные строительные блоки программ на C++. Они позволяют разбивать код на логические части, которые можно повторно использовать.

В этом разделе вы узнаете, как правильно объявлять и определять функции в C++.

Синтаксис объявления функции

Объявление функции (прототип) включает тип возвращаемого значения, имя функции и список параметров:

// Прототип функции
тип_возвращаемого_значения имя_функции(тип1 параметр1, тип2 параметр2, ...);

Примеры объявлений функций:

// Примеры объявлений функций
int sum(int a, int b); // Функция sum принимает два int и возвращает int
void printMessage();   // Функция без параметров, не возвращает значение
double calculateArea(double radius); // Принимает double, возвращает double

Определение функции

Определение функции содержит её реализацию (тело функции):

тип_возвращаемого_значения имя_функции(параметры) {
    // Тело функции
    // ...
    return значение; // Для функций, возвращающих значение
}

Пример определения функции:

// Определение функции sum
int sum(int a, int b) {
    int result = a + b;
    return result;
}

// Определение функции printMessage
void printMessage() {
    std::cout << "Hello, World!" << std::endl;
}

Вызов функции

Функция вызывается по имени с передачей необходимых аргументов:

#include <iostream>

// Объявление функции
int multiply(int x, int y);

int main() {
    int a = 5, b = 3;
    int result = multiply(a, b); // Вызов функции
    
    std::cout << "Result: " << result << std::endl;
    return 0;
}

// Определение функции
int multiply(int x, int y) {
    return x * y;
}

Параметры по умолчанию

C++ позволяет задавать значения параметров по умолчанию:

// Функция с параметрами по умолчанию
void printInfo(std::string name, int age = 18, std::string country = "Russia") {
    std::cout << "Name: " << name << ", Age: " << age 
              << ", Country: " << country << std::endl;
}

int main() {
    printInfo("Alice"); // Использует age=18, country="Russia"
    printInfo("Bob", 25); // Использует country="Russia"
    printInfo("Charlie", 30, "USA"); // Переопределяет все параметры
    
    return 0;
}

Важно

Параметры по умолчанию должны быть указаны только в объявлении функции (обычно в заголовочном файле), а не в определении.

Разделение объявления и определения

В C++ принято разделять объявление (в .h файле) и определение (в .cpp файле):

math_functions.h 
// Объявления функций
int add(int a, int b);
double divide(double a, double b);
math_functions.cpp 
#include "math_functions.h"

// Определение функции add
int add(int a, int b) {
    return a + b;
}

// Определение функции divide
double divide(double a, double b) {
    if (b == 0) {
        return 0;
    }
    return a / b;
}

Встроенные функции (inline)

Ключевое слово inline подсказывает компилятору, что функцию можно встроить в код:

// Встроенная функция
inline int max(int a, int b) {
    return (a > b) ? a : b;
}

int main() {
    int x = 10, y = 20;
    std::cout << "Max: " << max(x, y) << std::endl; // Может быть заменено на (x > y ? x : y)
    return 0;
}

Совет по использованию функций

Разделяйте объявления и определения функций между .h и .cpp файлами для лучшей организации кода. Это особенно важно в больших проектах.
