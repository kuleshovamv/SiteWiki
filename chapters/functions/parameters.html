Параметры и возвращаемые значения функций

Параметры и возвращаемые значения - ключевые аспекты работы с функциями в C++. Они определяют, как данные передаются в функцию и возвращаются из неё.

В этом разделе вы узнаете о различных способах передачи параметров и возврата значений из функций.

Передача параметров по значению

При передаче по значению функция получает копию аргумента:

void increment(int x) {
    x++; // Изменяется только локальная копия
}

int main() {
    int num = 5;
    increment(num);
    std::cout << num; // Выведет 5 (оригинал не изменился)
    return 0;
}

Примечание

Передача по значению безопасна, но может быть неэффективна для больших объектов, так как создается полная копия.

Передача параметров по ссылке

Передача по ссылке позволяет изменять оригинальные переменные:

void increment(int &x) {
    x++; // Изменяется оригинальная переменная
}

int main() {
    int num = 5;
    increment(num);
    std::cout << num; // Выведет 6 (оригинал изменился)
    return 0;
}

Константные ссылки

Константные ссылки позволяют избежать копирования и гарантируют, что объект не будет изменен:

void printLargeObject(const BigObject &obj) {
    // obj нельзя изменить, но копирования не происходит
    obj.print();
}

Передача указателей

Указатели также позволяют изменять оригинальные переменные:

void increment(int *x) {
    (*x)++; // Изменяем значение по указателю
}

int main() {
    int num = 5;
    increment(&num); // Передаем адрес переменной
    std::cout << num; // Выведет 6
    return 0;
}

Осторожно с указателями

При работе с указателями нужно быть уверенным, что они не являются nullptr и указывают на допустимую область памяти.

Возвращаемые значения

Функции могут возвращать значения любого типа:

// Возврат простого типа
int square(int x) {
    return x * x;
}

// Возврат ссылки
int& getElement(std::vector &vec, int index) {
    return vec[index];
}

// Возврат указателя
int* findValue(int *arr, int size, int value) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == value) {
            return &arr[i];
        }
    }
    return nullptr;
}

Возврат по значению, ссылке и указателю

Способ возврата Пример Когда использовать Опасности
По значению int func() Простые типы, маленькие объекты Копирование больших объектов
По ссылке int& func() Изменяемые объекты, избегание копирования Возврат ссылки на локальную переменную
По константной ссылке const int& func() Большие неизменяемые объекты Те же, что и для обычной ссылки
По указателю int* func() Динамическая память, опциональные значения Утечки памяти, висячие указатели

Возврат нескольких значений

В C++ есть несколько способов вернуть несколько значений из функции:

1. Через параметры по ссылке

void getMinMax(const int *arr, int size, int &min, int &max) {
    min = arr[0];
    max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] < min) min = arr[i];
        if (arr[i] > max) max = arr[i];
    }
}

int main() {
    int arr[] = {3, 1, 4, 1, 5, 9};
    int min, max;
    getMinMax(arr, 6, min, max);
    // min = 1, max = 9
}

2. Через структуру или кортеж

#include <tuple>

std::tuple<int, int, double> calculateStats(int a, int b) {
    return {a + b, a * b, static_cast<double>(a) / b};
}

int main() {
    auto [sum, product, quotient] = calculateStats(10, 4);
    // sum = 14, product = 40, quotient = 2.5
}

Совет

Для возврата нескольких значений в C++17 и новее удобно использовать structured binding (как в примере выше с кортежем).
