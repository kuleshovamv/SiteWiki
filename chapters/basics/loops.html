Циклы в C++

Циклы позволяют многократно выполнять блок кода, пока выполняется определенное условие. В C++ существует несколько видов циклов, каждый из которых подходит для разных сценариев.

Цикл while

Цикл while выполняет блок кода, пока условие истинно.

// Синтаксис
while (условие) {
    // Тело цикла
}

// Пример
int i = 0;
while (i < 5) {
    std::cout << "i = " << i << std::endl;
    i++;
}

Цикл do-while

Цикл do-while сначала выполняет блок кода, а затем проверяет условие. Гарантирует хотя бы одно выполнение тела цикла.

// Синтаксис
do {
    // Тело цикла
} while (условие);

// Пример
int j = 0;
do {
    std::cout << "j = " << j << std::endl;
    j++;
} while (j < 5);

Цикл for

Цикл for объединяет инициализацию, условие и изменение счетчика в одной строке.

// Синтаксис
for (инициализация; условие; изменение) {
    // Тело цикла
}

// Пример
for (int k = 0; k < 5; k++) {
    std::cout << "k = " << k << std::endl;
}

Цикл for на основе диапазона (C++11)

Упрощает перебор элементов контейнеров и массивов.

// Синтаксис
for (тип элемент : контейнер) {
    // Тело цикла
}

// Примеры
int arr[] = {1, 2, 3, 4, 5};
for (int num : arr) {
    std::cout << num << " ";
}

std::vector<int> vec = {10, 20, 30};
for (int val : vec) {
    std::cout << val << " ";
}

Операторы управления циклами

Оператор Описание Пример
break Немедленно завершает выполнение цикла if (i == 3) break;
continue Пропускает текущую итерацию и переходит к следующей if (i % 2 == 0) continue;
return Выход из функции (и всех вложенных циклов) if (error) return;

Вложенные циклы

Циклы могут содержать другие циклы, создавая вложенные конструкции.

// Вывод таблицы умножения
for (int i = 1; i <= 10; i++) {
    for (int j = 1; j <= 10; j++) {
        std::cout << i * j << "\t";
    }
    std::cout << std::endl;
}

Бесконечные циклы

Циклы могут быть бесконечными, если условие всегда истинно.

// Варианты бесконечных циклов
while (true) {
    // Бесконечный цикл
}

for (;;) {
    // Бесконечный цикл
}

Совет по выбору цикла

Используйте цикл for, когда известно количество итераций. Цикл while лучше подходит, когда количество итераций неизвестно заранее. Цикл do-while используйте, когда нужно гарантировать хотя бы одно выполнение тела цикла.
