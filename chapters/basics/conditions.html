Условные конструкции в C++

Условные конструкции позволяют управлять ходом выполнения программы в зависимости от выполнения определенных условий. В C++ существуют несколько способов организации ветвления кода.

Конструкция if-else

Самая распространенная условная конструкция в C++:

if (условие) {
    // выполняется, если условие истинно
} else {
    // выполняется, если условие ложно
}

Примеры использования

int age = 18;

// Простое условие
if (age >= 18) {
    std::cout << "Доступ разрешен" << std::endl;
} else {
    std::cout << "Доступ запрещен" << std::endl;
}

// Несколько условий
if (age < 13) {
    std::cout << "Ребенок" << std::endl;
} else if (age < 18) {
    std::cout << "Подросток" << std::endl;
} else {
    std::cout << "Взрослый" << std::endl;
}

Тернарный оператор

Короткая форма записи простых условий:

переменная = (условие) ? значение_если_истина : значение_если_ложь;

Пример использования

int a = 5, b = 10;
int max = (a > b) ? a : b;

std::cout << "Максимальное число: " << max << std::endl;

Конструкция switch-case

Используется для выбора одного из многих вариантов:

switch (выражение) {
    case значение1:
        // код для значение1
        break;
    case значение2:
        // код для значение2
        break;
    // ...
    default:
        // код, если ни один case не подошел
}

Пример использования

char grade = 'B';

switch (grade) {
    case 'A':
        std::cout << "Отлично!" << std::endl;
        break;
    case 'B':
        std::cout << "Хорошо" << std::endl;
        break;
    case 'C':
        std::cout << "Удовлетворительно" << std::endl;
        break;
    case 'D':
        std::cout << "Неудовлетворительно" << std::endl;
        break;
    case 'F':
        std::cout << "Не сдал" << std::endl;
        break;
    default:
        std::cout << "Неверная оценка" << std::endl;
}

Совет по использованию switch

Не забывайте использовать break после каждого case, иначе выполнение "провалится" в следующий case. Это может быть полезно в некоторых случаях, но чаще приводит к ошибкам.

Логические операторы

Для создания сложных условий используются логические операторы:

Оператор Описание Пример
&& Логическое И if (a > 0 && b > 0)
|| Логическое ИЛИ if (a > 0 || b > 0)
! Логическое НЕ if (!isFinished)

Пример сложного условия

int age = 25;
bool hasLicense = true;

if (age >= 18 && hasLicense) {
    std::cout << "Можно водить машину" << std::endl;
} else if (age >= 18 && !hasLicense) {
    std::cout << "Нужно получить права" << std::endl;
} else {
    std::cout << "Слишком молод" << std::endl;
}
