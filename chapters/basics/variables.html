Переменные в C++

Переменные - это фундаментальная концепция в программировании на C++. Они представляют собой именованные области памяти, которые хранят данные определенного типа.

В этом разделе вы узнаете, как объявлять, инициализировать и использовать переменные в C++.

Объявление переменных

В C++ переменные объявляются с указанием типа и имени:

// Объявление переменной
int age;
double price;
char letter;

Где:

  • int - целочисленный тип
  • double - тип с плавающей точкой двойной точности
  • char - символьный тип

Инициализация переменных

Переменные можно инициализировать при объявлении:

// Инициализация при объявлении
int count = 10;
double temperature = 36.6;
char grade = 'A';

В C++11 появилась универсальная инициализация с фигурными скобками:

// Универсальная инициализация (C++11)
int width{5};
double height{1.75};
std::string name{"Alice"};

Основные типы данных

C++ предоставляет несколько основных типов данных:

Тип Описание Размер (байт) Пример
int Целое число 4 42
float Число с плавающей точкой 4 3.14f
double Число с плавающей точкой (двойная точность) 8 2.71828
char Символ 1 'A'
bool Логический тип 1 true, false

Область видимости переменных

Область видимости определяет, где переменная может быть использована:

#include <iostream>

int globalVar = 10;  // Глобальная переменная

int main() {
    int localVar = 5;  // Локальная переменная
    
    {
        int blockVar = 3;  // Переменная блока
        std::cout << blockVar << std::endl;
    }
    
    // Ошибка: blockVar не видна здесь
    // std::cout << blockVar << std::endl;
    
    std::cout << globalVar << std::endl;
    std::cout << localVar << std::endl;
    
    return 0;
}

Константные переменные

Для создания неизменяемых переменных используют модификатор const:

const double PI = 3.14159;
const int MAX_USERS = 100;

// Ошибка: нельзя изменить константу
// PI = 3.14;

Автоматическое определение типа (C++11)

С C++11 можно использовать ключевое слово auto:

auto number = 42;      // int
auto price = 19.99;    // double
auto letter = 'A';     // char
auto isActive = true;  // bool

Правила именования переменных

Рекомендации по именованию переменных:

  • Имена должны быть описательными (userAge лучше чем ua)
  • Можно использовать camelCase или snake_case
  • Нельзя начинать с цифры
  • Избегайте зарезервированных слов (int, return и т.д.)
  • Константы обычно именуются в UPPER_CASE

Совет по использованию переменных

Всегда инициализируйте переменные при объявлении. Неинициализированные переменные содержат "мусор" и могут привести к неожиданному поведению программы.
